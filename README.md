Análisis de Eficiencia - Alura Store 📊
Este proyecto forma parte del desafío de Alura Store, donde el objetivo principal es asesorar al Sr. Juan sobre qué tienda de su cadena debe vender para financiar un nuevo emprendimiento. El análisis se centra en identificar la unidad con menor rendimiento mediante el uso de Python y ciencia de datos.

🎯 Objetivo del Desafío
Analizar los datos de ventas, rendimiento y reseñas de 4 tiendas para determinar cuál es la menos eficiente basándose en métricas objetivas.

🛠️ Herramientas Utilizadas
Python 3.x

Pandas: Carga y manipulación de datos CSV.

Matplotlib / Seaborn: Creación de visualizaciones dinámicas.

Colab Notebook: Entorno de desarrollo para el análisis.

📈 Análisis Realizado
Para llegar a una conclusión, se evaluaron los siguientes indicadores clave de rendimiento (KPIs):

Ingresos Totales: ¿Qué tienda aporta menos al flujo de caja?

Satisfacción del Cliente: Análisis de reseñas y feedback.

Logística: Tiempo y costo de envío promedio.

Mix de Productos: Categorías más vendidas y rotación de inventario.

📊 Visualizaciones
El proyecto incluye un análisis visual profundo con al menos 3 tipos de gráficos:

Gráfico de Barras: Comparativa de ingresos por tienda.

Gráfico Circular: Distribución de categorías más vendidas.

Gráfico de Dispersión / Líneas: Relación entre reseñas y volumen de ventas.

Nota: Las imágenes de los gráficos generados se encuentran en la carpeta /visualizaciones (o dentro del notebook).
